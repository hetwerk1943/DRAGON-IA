# UltraChat AI

## Opis
UltraChat AI to własna implementacja ChatGPT z frontendem, backendem, historią czatu i SAFE AdSense.

## Uruchomienie

1️⃣ Backend:
```bash
cd backend
cp .env.example .env
# Wstaw swój klucz OpenAI w .env
npm install
npm start
```

2️⃣ Frontend:
```bash
npx http-server frontend -p 8080
```
